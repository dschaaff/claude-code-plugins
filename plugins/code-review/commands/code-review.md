# Code Review Request

Review the recent code changes for quality, security, and adherence to best practices.

Use the **code-reviewer** agent to perform a comprehensive review of:
- Code quality and maintainability
- Security vulnerabilities
- Architecture and design patterns
- Test coverage
- Documentation completeness
- Performance considerations

The agent will provide structured feedback with categorized issues (Critical/Important/Suggestions) and actionable recommendations.
